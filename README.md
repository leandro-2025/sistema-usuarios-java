# Sistema de Cadastro de Usuários em Java

Projeto simples de cadastro de usuários desenvolvido em Java para praticar conceitos de programação orientada a objetos, listas e estrutura de menus no console.

## Funcionalidades

- Cadastrar usuário
- Listar usuários cadastrados
- Buscar usuário pelo nome
- Remover usuário
- Menu interativo no console
- Validação de entrada de dados

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git e GitHub

## Como executar o projeto

1. Clone o repositório

git clone https://github.com/leandro-2025/sistema-usuarios-java

2. Abra o projeto no IntelliJ

3. Execute a classe `Main.java`

## Objetivo do projeto

Este projeto foi criado para praticar:

- Classes e objetos
- ArrayList
- Estrutura de menus
- Organização de código em pacotes
- Boas práticas iniciais de Java
